# cipereusz premium

Next gen of [cipereusz](https://github.com/suchencjusz/cipereusz) (now with real AI!!1!)

![cipereusz-premium](readme_image.png)

## Setup

Create a `.env` file from `.env.example` and fill in:

- `TOKEN_DISCORD`
- `GROQ_API_KEY`
- `GROQ_CHAT_MODEL`
- `GROQ_VISION_MODEL`
- `DATABASE_PATH`
- `RANDOM_REPLY_CHANCE`
- `IDLE_SECONDS`
- `MEMORY_BATCH_SIZE`
- `MEMORY_RECENT_MESSAGES`
- `MENTION_REPLY_LIMIT`
- `ADMIN_USER_ID`
- `BOT_PING_ID`
- `BOT_PING_CHANCE`
- `TZ`

## Docker

```bash
docker compose up --build
```
